SkillSwap is a full-stack web application built using the MERN stack (MongoDB, Express.js, React.js, Node.js) that enables users to exchange skills with each other instead of traditional monetary transactions.



The platform is designed to connect people who want to learn a skill with those who are willing to teach, promoting collaborative learning and peer-to-peer knowledge sharing.

##### **System Architecture**



* Frontend (React) handles user interaction, skill browsing, and request management.
* Backend (Express + Node.js) exposes REST APIs for users, skills, and exchange requests.
* MongoDB stores user profiles, skills, and exchange relationships.
* Cloudinary manages profile images and media efficiently.
* The application is deployed on Render with environment-based configurations.
